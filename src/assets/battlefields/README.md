# Battlefield art

Size (2560 × 1440)

Drop landscape PNG, JPG, or WebP files in this folder. They are bundled into the
pre-battle horizontal rotation automatically. Use a universe id in the filename
(for example `marvel-new-york.png` or `mario-castle.webp`) to give that universe
the standard battlefield affinity boost without adding metadata.

Built-in arenas can define friendlier names, subtitles, colors, and multiple
affinities in `src/data/battlefields.ts`.

# Universe (franchise) art drop-in — slot reel 1

Put a PNG named after a franchise id here (e.g. `dc.png`, `marvel.png`,
`nintendo.png`, `star-wars.png`, `anime.png`, `toons.png` — ids are in
`src/data/franchises.ts`) and the UNIVERSE reel shows it automatically instead
of the text plaque.

- Recommended size: ~248×136 (2× the reel plaque) or larger; images are
  cover-fitted and cropped to a 124×68 rounded window.
- Franchises without art keep the colored text plaque.
